# Transparent Coupon — Honest Deals

A **privacy-first** Chrome extension that auto-applies coupon codes and tracks price history on Amazon & 1000+ stores. Built as the ethical alternative after the Honey scandal — **zero affiliate hijacking, zero dark patterns.**

## Features
- 🔍 Auto-detect coupons on 1000+ e-commerce sites
- ⚡ One-click auto-apply at checkout
- 📊 Price history tracking & drop alerts
- 🔒 No data collection, no tracking, no link hijacking
